## Getting Started

No dependencies or build steps needed. Just clone and open.

```bash
git clone https://github.com/YathavanJD/Yathavan_Portfolio.git
cd Yathavan_Portfolio
open index.html
```

Or serve it locally for a better dev experience:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.
